## Payment Methods That Work in Nepal

> PayPal does NOT work for receiving money in Nepal. Use these instead.

| Method | Best For | Setup |
|--------|----------|-------|
| **Payoneer** | Upwork, Fiverr, Direct Clients | Link with Nabil/Global IME bank |
| **Skrill + eSewa** | Freelancer.com users | Skrill -> eSewa -> Bank (1.45% fee) |
| **Wise** | Direct international clients | Client sends to your Nepali bank via SWIFT |
| **SWIFT Bank Transfer** | Large payments ($2000+) | 3-7 days, higher fees |
| **eSewa / Khalti** | Local clients, domestic work | Instant local transfers |
| **Dollar Card** | Paying for online tools | Nabil, Global IME, NMB bank ($500/year limit) |
